----------------- 模版 
=> 完善流程:
	列表-> 详情->添加->修改->删除
=> 列表:  通用模版  
	显示:   [ id 名称 状态 详情 修改 删除 ] 
	操作:  添加  查看详情 修改 删除
	操作的URL: 谁用 谁设定
=> 详情 通用模版  
	显示:   [ 字段名 字段值 ] 
	操作:  返回 修改
	操作的URL: 谁用 谁设定
=> 添加 通用模版  
	显示:   [ 需要添加的字段名 字段提示] 
	操作:  返回 提交
	操作的URL: 谁用 谁设定
=> 修改  通用模版  
	显示:   [ 字段名 字段值] 
	操作:  返回 编辑字段值 提交
	操作的URL: 谁用 谁设定
=> 删除  无模版  通用操作 
	操作: 删除数据 (可以删除数据,可以修改状态字段)
	操作的URL: 谁用 谁设定
